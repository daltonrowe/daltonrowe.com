Public Library Website Project

Prospectus 2021

Every public library deserves a great website.

Public libariries are key factors in technology and internet education, we should seek to give back to these amazing institutions. The Public Library Website Project is a charitable initiative to develop and deploy quality websites for any public library that needs one.

The Goal.

The Public Library Website Project seeks to do the following:

  • Develop and document a quality WordPress theme suited to public libraries.
  • Provide training resources and direct support to libraries.
  • Provide cost-effective hosting services for public library sites.

About Me.

Hey I'm Dalton Rowe. I've been making websites for the last 10 years. I've been lucky enough to make five public library websites, each and every one a great learning experience. Public libraries really use their websites, and any service supporting them needs to provide quality tools for all aspects of library websites.

My dream is to work hand in hand with local libraries, building a quality website framework to match their needs.
